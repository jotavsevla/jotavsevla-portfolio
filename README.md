# João Victor Araújo • Terminal Premium Portfolio

Landing page em React + Vite com foco em recrutadores para estágio backend.

## Stack

- React 19 + Vite + TypeScript
- CSS custom com tema dark/light
- GitHub API pública com cache local (`portfolio_cache_v1`)
- Vitest + Testing Library + Playwright

## Desenvolvimento

```bash
npm install
npm run dev
```

## Testes

```bash
npm run test
npm run test:e2e
```

## Build

```bash
npm run build
npm run preview
```

## Deploy na Vercel

- Build command: `npm run build`
- Output directory: `dist`
